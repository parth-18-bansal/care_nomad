# care_nomad
Running the Care Application using the nomad


1) Install the nomad: https://developer.hashicorp.com/nomad/install
2) create the nomad cluster: https://developer.hashicorp.com/nomad/tutorials/get-started/gs-start-a-cluster
3) edit the /etc/nomad.d/nomad.hcl file to define the docker desktop .sock file and allow volumes
